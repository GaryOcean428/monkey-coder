Welcome to the Monkey Coder Core Orchestration Engine API. This is the backend service that powers AI-driven code generation and analysis.

Available Endpoints

GET /health - Health check
GET /metrics - Prometheus metrics
POST /v1/execute - Execute AI task
GET /v1/billing/usage - Get usage metrics
GET /v1/providers - List AI providers
GET /v1/models - List available models

Getting Started

To use this API, you'll need an API key. Install the CLI or SDK:

# Install CLI
npm install -g monkey-coder-cli

# Install Python SDK
pip install monkey-coder-sdk

# Install Core Library
pip install monkey-coder-core
